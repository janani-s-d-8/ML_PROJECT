# ML_PROJECT

# 📊 Customer Satisfaction Prediction (CSAT) - ML Project

## 🧠 Project Overview
This project focuses on predicting **Customer Satisfaction (CSAT)** using machine learning techniques based on customer interaction and service-related features.

The goal is to build a classification model that helps understand customer experience and improve service quality.
